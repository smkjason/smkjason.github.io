My Projects

Fall 2019

Jarvis

This is a native Android App that finds the best time that works for the most people for a meeting based on their schedule on Google Calendar. I intergrated Google Sign-In. Designed the UI by using XML and Java. Implemented HTTP Request for all communication with the backend.
Developed with Android Studio.

Fall 2020

Cache Simulation

This is a project that models and simulates different types of caches to study and anlayze its effects on CPU performance. The types of caches implemented include Direct-Mapped Cacheing, n-way Set Associative Cacheing, and Victim Caches. I've also utilized GDB Debugger tool to help me out.
This project can only be run on Linux OS.

January 2020

InGRADEient

This is a mobile app that analyzes the ingredients of toiletries and gives useful information on its environmental-friendliness and health-related information. The list of ingredients is captured through phone camera which is then analyzed to give its verdict. We used Google Text Recognition API to read the picture. I've implemented the UI of taking a picture through the app with React Native and Expo.
